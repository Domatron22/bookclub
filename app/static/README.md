# Static Assets

This directory contains static files served by the application.

## Structure

```
static/
├── css/
│   └── custom.css      # Custom CSS styles
├── js/
│   └── main.js         # Main JavaScript file
└── images/             # Place images here (coming soon)
```

## Adding Custom Styles

Edit `css/custom.css` to add custom styles that aren't covered by Tailwind CSS.

## Adding JavaScript Functionality

Edit `js/main.js` to add interactive features. The file already includes:
- Notification system
- Club code clipboard copy
- Confirmation dialogs for destructive actions
- Auto-uppercase for club code inputs

## Future Enhancements

- Add a favicon.ico file in this directory
- Add custom images for branding
- Add book cover placeholder images
